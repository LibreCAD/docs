.. _widgets: 

Dock Widgets
=============

Dock widgets serve two purposes:

    1. In additional to the menus and toolbars, provides another method to access :ref:`drawing tools <tools>`:

    - Circle
    - Curve
    - Dimension
    - Ellipse
    - Info
    - Line
    - Modify
    - Polyline


    2. Provides additional tools:

    - Block List
    - Command Line
    - Layer List
    - Library Browser
    - Pen Wizard
